# 蜡笔优课功能对标方案

## 一、蜡笔优课核心功能分析

### 1.1 核心功能模块

| 模块 | 功能描述 | 对应我们的组件 |
|--------|---------|--------------|
| 智能排课 | 自动检测时间冲突，优化资源配置 | Schedule |
| 学员管理 | 学员档案、课时记录 | StudentsPage |
| 课时管理 | 课时购买、消耗、结算 | ParentDashboard |
| 考勤管理 | 一键考勤、上课提醒 | CheckInPage |
| 家校互动 | 消息通知、课堂点评 | EnhancedMessageCenter |
| 教学资源 | 课程资料、教学素材 | MaterialsPage |
| 作业管理 | 作业布置、提交、批改 | HomeworkPage |
| 学习进度 | 课程完成记录、技能评级 | StudentProgressPage |
| 作品集 | 学员作品展示 | PortfolioPage |

### 1.2 蜡笔优课的 UI/UX 特点

- 简洁明快：界面设计清晰，色彩搭配舒适
- 卡通插画风格：生动活泼的卡通元素
- 易于操作：用户界面友好，功能布局合理
- 活泼可爱：使用珊瑚红、青绿色、明黄色等活泼颜色
- 科技感强：融入 AI 技术和大数据分析
- 圆角设计：大量使用圆角卡片
- 渐变效果：按钮和背景使用柔和渐变
- 动画效果：微妙的悬停和点击动画

## 三、UI/UX 优化建议

### 3.1 配色方案

当前应用已采用蜡笔优课风格的配色：

- 主色：#FF6B6B（珊瑚红）
- 次色：#4ECDC4（青绿色）
- 强调色：#FFE66D（明黄色）
- 成功色：#95E1A3（浅绿色）
- 警告色：#FFB347（橙色）
- 信息色：#74B9FF（天蓝色）
- 紫色：#A29BFE（深紫色）
- 粉色：#FD79A8（粉红色）

### 3.2 组件优化建议

#### 3.2.1 卡片设计

- 使用 `card-crayon` 类替代普通卡片
- 添加渐变边框效果
- 增加圆角（rounded-3xl）
- 添加柔和阴影

#### 3.2.2 按钮设计

- 使用 `btn-crayon` 类替代普通按钮
- 添加渐变背景
- 增加悬停缩放效果
- 添加点击反馈动画

#### 3.2.3 图标和装饰

- 添加浮动的星星、爱心等装饰元素
- 使用 `icon-bounce` 类添加弹跳效果
- 添加 `pulse-dot` 类添加脉冲动画

#### 3.2.4 文字和排版

- 使用 `text-crayon` 类突出重要文字
- 增加渐变文字效果
- 优化字体大小和行高

### 3.3 交互优化

#### 3.3.1 动画效果

- 页面加载：使用 `animate-fade-in`
- 卡片进入：使用 `animate-slide-in`
- 缩放效果：使用 `animate-scale-in`
- 悬浮效果：使用 `animate-float`

#### 3.3.2 反馈机制

- 按钮点击：使用 `btn-press:active`
- 悬停效果：使用 `card-hover:hover`
- 过渡动画：使用 `transition-smooth`

## 四、具体优化建议

**当前状态**：
- 支持课程创建、编辑、删除
